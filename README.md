# Setting up the Database

1. cd into `./server`

2. In `./server` modify `DATABASE_URL` in `.env` so that it uses your postgres user and password

3. Run `npm run yarn:m`

# Running Server

1. cd into `./server`

2. Install dependencies `npm install`

3. Run `node app.js`

# Running Client

1. cd into `./client`

2. Install dependencies `npm install`

3. Run `ng serve`

4. Navigate to `http://localhost:4200`
